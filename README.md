A backend URL Shortener application built using Spring Boot and Maven.
The system generates unique short URLs for long links and handles URL mapping using in-memory storage.

 Features

REST API-based URL shortening

Unique short code generation

In-memory URL mapping (HashMap)

Maven-based project structure

Modular and scalable backend architecture

 Technologies Used

Java

Spring Boot

Maven

REST APIs

HashMap (Data Structures)

 API Flow

Client sends long URL via REST endpoint

Server generates unique short code

Short code is mapped to original URL

(Future Scope) Redirect using HTTP 302 response
